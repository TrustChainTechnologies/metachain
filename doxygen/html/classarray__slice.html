Detailed Description

template<typename Iterable>
class array_slice< Iterable >

An read-only STL-style wrapper for array-style collections.

This class allows std::array, std::vector, std::string, and c-style arrays to be used interchangeably in functions that expect an iterable container. This is not a substitute for move overloads. Consider also that a contiguous buffer (c-style array) is more performant than the iterator abstraction.
