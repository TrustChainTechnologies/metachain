MCP (MetaChain Protocol)

A quick overview of the different MCPs

  • MCP01: Account and Wallet handling
  • MCP02: MetaChain and SubChain handling
  • MCP03: Blocks and Transaction handling
  • MCP04: Proof of Processes (PoS, PoW, PoT etc.)
  • MCP05: Workflow handling (Currency Transaction, Data Transactions etc.)
  • MCP06: Transaction Scripts and Signatures
  • MCP39: based on BIP39 mnemonic generation
